# wheels_on_lease

> A very basic Flutter project for renting vehicles in a given city. 

## features

Firebase email authentication<br/>
- Main Screen UI<br/>
- Profile Screen<br/>
- Fetch vehicles list from cloud db acc. to city<br/>
- Scrollable List of Vehicle tiles<br/>
